    🔑 配置 GitHub Token

    ×

    ⚠️ 重要安全说明

    为了防止 Token 被 GitHub 自动撤销，此系统采用安全配置方式：

    ⚠️ Token 只在当前浏览器会话中保存，页面刷新后需要重新输入

    如何获取 GitHub Token？

    1. 访问 GitHub Settings → Developer settings → Personal access tokens
    2. 点击 "Generate new token (classic)"
    3. 只勾选 "gist" 权限
    4. 复制生成的 Token

    注意：此安全设计避免了 Token 被意外提交到代码库而被撤销。
